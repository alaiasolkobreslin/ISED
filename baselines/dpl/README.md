# DeepProbLog

## Requirements
Run the following:

1. `cd baselines/dpl`

2. Install additional dependencies inside the virtual environment: `bash setup_dpl.sh`

## Experiments
To reproduce results in the paper,
```bash
cd baselines/dpl
python <TASK>/<FILE>.py
```
